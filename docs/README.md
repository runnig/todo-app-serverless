## App

A simple TODO app

## Functionality

Http API:
* Log in/Log out
* List TODO entries
* Create new TODO entries
* Remove an entry
* Update an entry
* Mark as done

## UI

Responsible Web UI.

## Development

* Should be testable locally: create a local database, initialize schema, populate sample data,
create new users, publish new TODO entries, etc
* Integration tests should be runnable against a local instance and a remote
  test instance
